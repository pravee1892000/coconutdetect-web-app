About App

Why We Developed It

In our village, we are coconut fruit harvesters. Identifying the exact ripeness of coconuts for harvesting has always been a challenge. To address this issue, I took it upon myself to develop a robust detection system. This system will help young starters easily identify the ripeness of coconuts.

How It Works

Our application uses YOLO (You Only Look Once), a state-of-the-art real-time maturity detection system. The app allows you to upload images or videos, or use your webcam to detect in real-time.

Technologies Used

  • Backend: Flask (Python)
  • Object Detection: Machine Learning, Ultralytics YOLO
  • Image Processing: OpenCV, PIL
  • Frontend: HTML, CSS, JavaScript

Features

  • Image coconut maturity detection
  • Video coconut maturity detection
  • Real-time webcam coconut maturity detection
  • Support for three classes
